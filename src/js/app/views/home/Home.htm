Quickstart

1. Add ratel.jar to your classpath.

2. Add RatelFilter to your web.xml and map it for all (/*) requests:

<?xml version="1.0" encoding="UTF-8"?>
<web-app>
    <filter>
        <filter-name>ratelFilter</filter-name>
        <filter-class>com.google.ratel.RatelFilter</filter-class>
        <init-param>
            <param-name>packageNames</param-name>
            <param-value>com.mycorp.service</param-value>
        </init-param>
    </filter>
    
    <filter-mapping>
        <filter-name>ratelFilter</filter-name>
        <url-pattern>/*</url-pattern>
    </filter-mapping>
</web-app>

3. Create a Ratel service to handle incoming requests (A Ratel service is a POJO annotated with @RatelService:

package com.mycorp.service;

  @RatelService
  public class MyService {

    public Sting myMethod() {
  
      var json = "{\"message\": \"Hello World\"}";
      return json;
    }

  }
Ratel maps this service to the URL: 
http://localhost/MyService/MyMethod
