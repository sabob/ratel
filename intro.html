3. Create a Ratel service to handle incoming requests (A Ratel service is a POJO annotated with @RatelService:

package com.mycorp.service;

  @RatelService
  public class MyService {

    public Sting myMethod() {
  
      var json = "{\"message\": \"Hello World\"}";
      return json;
    }

  }
Ratel maps this service to the URL: 
http://localhost/MyService/MyMethod
